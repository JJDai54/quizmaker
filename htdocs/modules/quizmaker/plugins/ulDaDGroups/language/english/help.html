ulDadGroups : Regrouper des expressions par thème

Description :

Ce slide présente une liste d'items qu'il faut replacer dans les bons groupes.
Il peut y avoir jusqu'à quatre groupes.
Les items se déplacent par un "Drag And Drop".
Les items seront automatiquement mélanger dans les slides.

Objectif :

Replacer les expressions dans les groupes prédéfinis.

Options :

  • points : score obtenu quand l'association des couples est correct.
  • largeur des items : largeur maximum des items dans les groupes. attention de ne pas mettre une valeur trop grande au risque de perturber l'affichage.
  • libellés des groupes : il est possible de définir jusqu'à quatre groupes. Le nom des groupes sera repris dans la liste des proposition plus bas la prochaine réédition du slide. Seul seront pris en compte les groupes ayant un libellé.
  • couleur de fond des groupes : chaque groupe peut avoir une couleur différente, attention de ne pas choisir des couleurs trop foncées.
  • groupe par défaut : au lancement du quiz les expressions seront répartie soit dans tous les groupes définis, soit dans le groupe choisi par défaut.
  • disposition :  indique la disposition des groupes. Important : il faut choisir une disposition qui correspond au nombre de groupes définis.

Propositions :

Chaque proposition comprend :
  • expression : mot ou expression à replacer dans le bon groupe.
  • couleur de fond : chaque items peut avoir une couleur de fond différentes pou par exemple tromper l'utilisateur.
  • poids : ordre d'affichage lors de l'édition du slide. Cela n'a aucun impacte sur la répartition des items dans les groupes.
  • points : le nombre de points doit être au minimum de un. Les items dans les mauvais groupes  diminueront d'autant le score.
  • Groupe : liste déroulante des groupes.  Après avoir enregistré le libellé des groupe, ceux-ci apparaîtront dans cette liste lors de la prochaine édition du slide.
Il est possible de définir jusqu'à 16 propositions.
Pour supprimer un item, vider la zone de saisie
