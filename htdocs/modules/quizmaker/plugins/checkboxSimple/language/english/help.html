checkboxSimple : Multiple choice question

This slide offers several options in the form of checkboxes.
To each proposal you must assign a number of positive points for the correct answers, zero or negative for the wrong answers.
If the number of global points (areas of optins) is greater than zero, it is the latter which will be taken into account, otherwise it will be the sum of the items selected.

You can specify a maximum of 12 terms.
To delete an item, empty the corresponding text area, there is no need to shift them, they will be displayed correctly on the next modification.

Options :
  • Images:  you can associate an image which will be displayed on the left of the list of options
  • Image height : Defines the size of the image by the height, beware of images that are too large.
  • Shuffle the proposals : the list will be shuffled each time the quiz is launched.
