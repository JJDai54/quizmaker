Ce slide présente 1 à 3 listes.
Dans le cas d'une seule liste il s'agit d'éliminer les items qui ne sorrespondent pas aux critères énoncés dans la question.
Sil il y a plusueurs listes il s'agit de regrouper les items selon les critères correspondant aux titres des listes
Dans le cas ou il y a 3 listes, les items sont déplacés sur la liste suivante et pour la troisième vers la première liste.. Il faudra donc peut-être déplacer plusieurs fois un item pour l'amener dans le bon groupe.
Selon les options de paramètrage il faudra faire un click ou un double clique pour déplacer les items.
