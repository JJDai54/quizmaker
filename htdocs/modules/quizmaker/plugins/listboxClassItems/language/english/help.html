listboxClassItems : dispatcher les items dans les listes

Description :

Une, deux ou trois listes.
Plusieurs termes ou expression à replacer dans la bonne liste.
Pour obtenir le score il faut que tous les items soient à leur place, sinon la score sera de zéro.
Attention la gestion des points est différente selon qu'il y a une ou plusieurs listes (voir plus bas)

Objectif :

  • Une seule liste : il faut éliminer les items qui ne correspondent pas à la question.
  • Deux ou trois listes : il faut placer chaque item dans la bonne liste.

Options :

  • Points : Si (points == 0) alors c'est le nombre de points affectés aux bonnes et mauvaises.
  • Mélanger les propositions : la liste des items sera mélangée à chaque lancement du quiz.
  • Action sur clique souris : un simple ou double click permet de déplacer un item dune liste à une autre. Dans le cas dune liste litem  est supprimé, il nest pas possible de revenir en arrière. Dans le cas de deux ou trois listes, litem est déplacé dans la liste suivante, ou dans la première liste.
  • Groupes : Il faut définir au moins un groupe. Chaque défini correspond à une « listebox ». Pour définir un groupe il suffit de lui donner un nom. Ce nom sera repris dans la liste groupes des propositions après avoir enregistré et réédité la question.
  • Répartition : Dans le cas d'un seul groupe ça na aucun effet, tous les items seront dans la seule liste définie. Dans le cas de deux ou trois groupes les items sont répartis soit de manière aléatoire dans les groupes définis, soit dans le groupe choisi.


Réponses :

  • Un seul groupe : Indiquez pour chaque proposition correcte un nombre de points supérieur à zéro. Les mauvaises  propositions doivent avoir un nombre de points égal à zéro.
  • Deux ou trois groupes :
    • Indiquez pour chaque proposition correcte ou non un nombre de points supérieur à zéro et le groupe dans lequel elles doivent être placées.
    • Tout item qui ne sera pas dans le bon groupe forcera le score à zéro.
    • Dans le cas de plusisurs groupes, les proposition dont le nombre de points est égal à zéro sera forcé à 1.
