listboxSortItems  : Sort List

This slide presents a list of items that must be sorted using the side buttons.
It is the weight that defines the order of the correct solution. If an item has been forgotten, simply add it and modify its weight accordingly, the display will be refreshed on the next modification.

To delete an item, empty the input area

Important: Dance slide the number of points is defined at the question level (higher in the form).

Options :

  • Strict order: yes: order defined by weight - no: the order can be reversed


Example the colors of the rainbow:
  Strict order : "Red, Orange, Yellow, Green, Blue, Indigo, Purple"
  Non-strict order : "Red,Orange,Yellow,Green,Blue,Indigo,Purple" or "Purple,Indigo,Blue,Green,Yellow,Orange,Red,"

  • Title : It is possible to add a title which will be displayed just above the list.
  • Color of the buttons : Allows, depending on the chosen theme for the quiz, to select a more visible color for the side buttons.
  • Button height : allows you to define the height of the side buttons.
