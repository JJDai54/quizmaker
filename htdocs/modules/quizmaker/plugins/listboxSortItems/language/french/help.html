listboxSortItems : Trier les éléments

Description :

  • une question de tri, par exemple : donner l'ordre des couleurs de l'ar en ciel
  • une liste style listbox qu'ilfaut trier
  • Des boutons pour déplacer les éléments dans la liste
  • une image optionnelle
  • un titre pour pour aider l'utilisateur

Objectif :

Il faut ordonner la liste des élément selon l'ordre stipuilé dans la question à l'ai des boutons sur le côté.
L'ordre peut être inverse selon les options prises, voir plus bas.

Options :

  • points : score obtenu quans l'ordre est correct.
  • images (optionnelle): vous pouvez associer une image qui sera affichée selon le template qui sera sélectionné
  • hauteur de l'image : Défini la taille de l'image par la hauteur, attention aux images trop larges.
  • tri : il peut être strict ou inverse. Si il est strict ia réponse doit correspondre exactement à l'ordre dans le quel les propositions sont définies.
  • Titre : il est affiché juste au dessus des listes et peu aider l'utilisateur.
  • Couleur des boutons : plusieurscouleur sont à disposition, choisisez la plus adpater en fonction du thème choisi.
  • Hauteur des boutons : La hauteur des boutons latéraux peux être augmentée si besoin.

Popositions :

Liste de réponses. elle doivent être dans l'ordre demandé dans la question.
Le score est le nombres popints définis dans les options plus haut.
