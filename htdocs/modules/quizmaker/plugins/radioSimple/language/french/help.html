radioSimple : Question à réponse unique

Description :

  • une question à choix unique, mais plusieurs réponses peuvent être valides
  • une liste optionnelle termes ou d'expressions pour aider au choix des réponses
  • une image optionnelle
  • une liste de propositions

Objectif :

Il faut cocher une ou plusieurs propositions. Une liste de termes ou d'expression peut être définie pour aider l'utilisateur (voir "famille de mots").

Options :

  • points : Si (points == 0) alors c'est le nombre de points affectés aux bonnes et mauvaises réponses qui seront comptées. Si points > 0, c'est cette valeur qui sera utilisée pour le score.
    Dans tous les cas il faut affecter un nombre de points aux bonnes réponses pour les identifier et calculer le bon score.
  • images (optionnelle): vous pouvez associer une image qui sera affichée selon le template qui sera sélectionné
  • hauteur de l'image : Défini la taille de l'image par la hauteur, attention aux images trop larges.
  • mélanger les propositions : la liste des caractère alphanumériques ou expression sera mélangée à chaque lancement du quiz..
  • famille de mots (optionnelle) : Il s'agit d'une liste de termes ou d'expression qui on des points communs avec les bonne réponses. Elle doit être séparée par une virgule ","'

Réponses :

Liste de réponses. Chaque réponse peut avoir un nombre de points inférieurs égaux ou supérieurs à zéro.
Nombre de points inférieur à zéro : le nombre de points est défalqué du total.
Nombre de points égal à zéro : le score sera de zéro.
Nombre de points supérieur à zéro : il peut y avoir plusieurs bonnes réponses, mais seule celle cochée sera prise en compte pour le score. Cela peut être intéressant pour mettre en valeur les réponses plus difficiles.
Le poids permet de réordonner la liste sans devoir faire des "couper / coller". Si l'option "Mélanger les propositions" a été cochées l'ordre des réponses à peu d'importance.
Pour supprimer un item vider la zone de texte correspondante, inutile de les décaler, ils seront affichés correctement à la prochaine modification.
Si le nombre de points global (Zones des options) est supérieur à zéro, c'est ce dernier qui sera pris en compte sinon ce sera la somme des items sélectionnés.
