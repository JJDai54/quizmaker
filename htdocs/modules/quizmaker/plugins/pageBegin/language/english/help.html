pageBegin : page de présentation du quiz

Description :

  • deux zones de textes pour informer lutilisateur
  • une image optionnelle
Cette page est automatiquement ajoutée lors de la création d'un quiz et ne peut pas être supprimée.

Objectif :

Présenter le quiz, et éventuellement donner quelques indication sur celui-ci.

Options :

  • images : elle sera affichée au-dessus des deux zones de texte.
  • height : hauteur de l'image. Attention de ne pas ajouter des images trop lourdes au risque de ralentir l'affichage.
  • zones de texte 1 (optionnelle): une zone de texte qui peut incorporer des variables du quiz (nombre de question, résultat, sommaire, ...)
  • zones de texte 2 (optionnelle): une zone de texte qui peut incorporer des variables du quiz (nombre de question, résultat, sommaire, ...)
L'éditeur utilisé pour les zones de texte est celui paramétré dans les options du module.

Propositions :

Deux zones de texte libres.
Il est possible d'insérer les variables du quiz ci-dessous  :
  • "Code" : Code a insérer dans le texte, ces codes doivent être entre accolades.
  • "Pages"' : Indique sur quelle pages il est préférable d'utiliser ces codes.
  • "Description" : Description des valeurs de remplacement.
