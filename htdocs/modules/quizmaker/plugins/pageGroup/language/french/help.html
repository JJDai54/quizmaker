PageGroup - Page de regroupements des questions par thème

Ce slide permet de définir une ou plusieurs pages intermédiaires pour regrouper les questions par thème.
Il est constitué de deux zones de texte qui peuvent être utilisées ensemble ou indépendamment.
L'éditeur utilisé est celui paramétré dans les options du module.
Il est possible d'afficher les résultats intermédiaures en utilisant les balises ci-dessous qui seront remplacées lors de son affichage par les valeurs obtenues:


Balise
Description
{repondu}

Nombre de réponses données. Les questions sans réponses ne sont pas comptabilisées, cependant certaines sont automatiquement  prises en compte.
{totalQuestions} Nombre total de questions du quiz
{score} Score total  obtenu. Ce score peut être négatif selon les points donnés à chaque proposition des questions.
{scoreMaxiQQ} Score maximum possible
{scoreMiniQQ} Score minimum possible. Ce score peut être négatif selon les points donnés à chaque proposition des questions.
{duree}

Temps d'exécution du quiz depuis la première question jusquà l'affichage du résultat.

Important :  chaque balise est entourée d'accolades et doit être en minuscules.

Exemple :
- Nombre de réponses faites : {repondu} / {totalQuestions}
- Votre score est de {score} / {scoreMaxiQQ} (score minimum : {scoreMiniQQ}) 
- Votre temps de réponse est de {duree}

Le score peut être négatif selon les points accordés aux propositions de chaque question.
