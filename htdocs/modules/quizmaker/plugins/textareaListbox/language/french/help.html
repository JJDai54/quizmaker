textareaListbox : Retrouvez les mots manquants

Description :

  • un texte contenant des mots entre accolades qu'il faut trouver.
  • plusieurs liste sous forme de liste déroulante qui correspondenet aux mots à trouver.
  • une liste optionnelle de mots  qui n'ont pas de rapport aec le texte.

Objectif :

Les mots entre accolages ont été remplacés par des chiffres entre accollades.
Il faut retrouver les mots manquants en les sélectionant dans les liste déroulante correspondantes aux chiffres indiqués.
Le même mot peut apparaitre plusieurs dans le texte, il comptera pour un seul.
Le texte est insenssible à la casse.
Il peut s'agire de mots ou d'expression à trouver.

Options :

  • points : score obtenu quans l'ordre est correct.
  • oriantation :
    • Horizontale : le texte et les listbox sont l'un à côté de l'autre
    • Vertical : les listbox sont au-dessous du texte.
  • Pointspar mot trouvé :  Chaque mot remis à sa place compte pour autant de points.
  • Points par mauvais mot : Il faut définir plus une liste de mots étrangés au texte.. Chaque mots erronés remplacé dans le texte diminue le score d'autant.
  • coulaur des balises : permet de choisit la couleur des chiffres mis entre accolade pour une meilleure lisibilité selon le thème choisi.
  • Texte : texte a reconstituer. Placez entre accolades les mots à trouver.
    exemple de texte avec les mots à trouver entre accolades, ici : toujours, miroir, lame, esprit:

    Homme libre, {toujours} tu chériras la mer !
    La mer est ton {miroir} ; tu contemples ton âme
    Dans le déroulement infini de sa {lame},
    Et ton {esprit} n'est pas un gouffre moins amer.

Propositions :

Cette liste est optionnelle et est destinée à tromper l'utilisateur. Elle sera ajouté aux mots entre accolades dans les liste déroulante .
Par exemple avec le texte de baudelaire ci-dessus, il pourrait être interessant d'ajouter les mots : jamais, titoir, mouroir, rame, flamme, palme, ....
Le score est le nombres popints définis dans les options lus haut.
