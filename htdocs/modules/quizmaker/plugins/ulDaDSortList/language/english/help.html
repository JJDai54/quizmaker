ulDaDSortList : Sort a list of terms

This slide presents a list of items that must be sorted.
The displacement of the items will be done by a "Drag And Drop".
The order can be strict or soft, ie it can be reversed.

It is the weight that defines the order of the correct solution. If an item has been forgotten, simply add it and modify its weight accordingly.
The list will be displayed in order on the next modification.

The items will automatically shuffle into the javascript slides.

To delete an item, empty the input area

Options:
  • Strict order: yes: order defined by weight - no: the order can be reversed

Example the colors of the rainbow:
     Strict order : "Red, Orange, Yellow, Green, Blue, Indigo, Purple"
     Non-strict order: "Red,Orange,Yellow,Green,Blue,Indigo,Purple" or "Purple,Indigo,Blue,Green,Yellow,Orange,Red,"

  • Title : It is possible to add a title which will be displayed just above the list.
