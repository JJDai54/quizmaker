comboboxSortItems : Trier les éléments

Description :

  • une question de tri, par exemple : donner l'ordre des couleurs de l'arc en ciel
  • plusieurs listes déroulantes qui reprennent chacune la liste des éléments à trier.
  • une image optionnelle
  • un titre pour pour aider l'utilisateur

Objectif :

Il faut dans chaque liste sélectionner un élément de telle sorte à afficher les éléments dans l'ordre stipulé dans la question.
L'ordre peut être inverse selon les options prises, voir plus bas.

Options :

  • points : score obtenu quand l'ordre est correct.
  • images (optionnelle): vous pouvez associer une image qui sera affichée selon le template qui sera sélectionné
  • hauteur de l'image : Défini la taille de l'image par la hauteur, attention aux images trop larges.
  • tri : il peut être strict ou inverse. Si il est strict la réponse doit correspondre exactement à l'ordre dans le quel les propositions sont définies.
  • Titre : il est affiché juste au dessus des listes et peu aider l'utilisateur.

Propositions :

Liste de réponses. elles doivent être dans l'ordre demandé dans la question.
Le score est le nombres points définis dans les options lus haut.
