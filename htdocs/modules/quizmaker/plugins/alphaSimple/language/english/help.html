alphaSimple: Character question

Description:

  • a question that asks about alphanumeric characters or expressions.
  • Words that relate to the question can be enclosed in braces. In this case they will be included in the slide according to the selected template. THE braces will be automatically removed.
  • an image (optional)
  • a directive to help the user
  • a list of alphanumeric characters or several expressions clickable

Objective:

You must select one or more alphanumeric characters or expression that matches the question

Options:

  • points: If (points == 0) then this is the number of points allocated to good and bad answers which will be counted. If points > 0, it is this value which will be used for the score.
  • in all cases it is necessary to assign a number of points to the correct ones answers to identify them and calculate the correct score.
  • images (optional): you can associate an image that will be displayed depending on the template that will be selected
  • image height: Defines the size of the image by the height, Be careful with images that are too wide.
  • mix the propositions: the list of alphanumeric characters or expression will be mixed each time the quiz is launched.
  • directive (optional): a short text to indicate to the user what he should do
  • proposals: List of alphanumeric characters or expressions clickable. Each character or expression must be separated by a hyphen "-". To force a newline in the sequence of characters or expression just double the separator.
  • “X” button:   Clear the proposal field
  • “@” button: Assigns one of the characters to the proposal field alphabetical
  • “#” button: Assigns one of the characters to the proposal field digital
  • "@#" button: Assigns a list of characters to the proposal field alphabetical and numerical
  • Template: the slide offers several layouts:
    • the three short lines and  black represent the words between braces included in the question
    • the red shape represents the answer(s) clicked by the user
    • The blue shape represents the image
    • the three long lines represent the character proposal alphanumeric

Proposals:

List of correct answers. Each answer must have a number of points greater than zero.
It is possible, although not recommended, to add wrong answers with a negative number of points.
The weight allows you to reorder the list without having to cut / stick. If the "Mix proposals" option was checked, the order of the answers of little importance.
