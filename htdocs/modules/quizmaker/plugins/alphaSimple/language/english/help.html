alphaSimple: Character question

Description:

  • a question that asks about alphanumeric characters or expressions.
  • Words that relate to the question can be enclosed in braces in the wording of the question.
    They will be included in the slide according to the selected template.
    The braces will be automatically removed.
  • an image (optional)
  • a directive to help the user
  • a list of alphanumeric characters or several clickable expressions.

Objective:

You must select one or more alphanumeric characters or expression that matches the question

Options:

  • points: this value can be defined in two places:
    at the question level:  If the value is greater than zero, This is the value that will be counted in the event of a correct response, otherwise these are the points defined for each proposal which will be taken into account.
    at the level of the propositions: the correct answers must have a value greater than zero. The wrong answer may have a value equal to or less than zero.
  • in all cases it is necessary to assign a number of points to the correct ones answers to identify them and calculate the correct score.
  • images (optional): you can associate an image which will be displayed according to the template that will be selected
  • image height: Defines the size of the image by the height, Be careful with images that are too wide.
  • mix the propositions: the list of characters alphanumeric or expression will be mixed each time the quiz is launched.
  • directive (optional): a short text to indicate to the user what to do
  • proposals: List of alphanumeric characters or expressions separated by a hyphen "-".
    To force a newline in the sequence of characters or expressions just double the separator.
  • “X” button:   Clear the proposal field
  • "@" button: Assigns to the proposal field a list of alphabetic characters
  • "#" button: Assigns to the proposal field a list of numeric characters
  • button "@#": Assigns to the proposal field a list of alphabetic and numeric characters
  • Template: the slide offers several provisions:
    • the three short lines and  black represents the words in curly brackets included in the question
    • the red shape represents the answer(s) clicked by the user
    • The blue shape represents the image
    • the three long lines represent the proposition of alphanumeric characters

Proposals:

List of correct answers: Each answer must have a number of points greater than zero.
List of wrong answers: Each wrong answer must have a number of points equal to or less than zero.

The weight allows you to reorder the list without having to cut / paste.
If the “Mix proposals” option was checked, the order of the proposals do not matter.
