alphaSimple : Question de caractères

Description :

  • une question qui porte sur des caractères alphanumériques ou des expressions.
  • Les mots qui portent sur la question peuvent entre accolade. Dans ce cas ils seront repris dans le slide selon le template sélectionné. Les accolades seront automatiquement supprimées.
  • une image (optionnelle)
  • une directive pour aider l'utilisateur
  • une liste de caractères alphanumériques ou de plusieurs expressions cliquables

Objectif :

Il faut sélectionner un ou plusieurs caractères alphanumériques ou expression qui correspondent à la question

Options :

  • points : Si (points == 0) alors c'est le nombre de points affectés aux bonnes et mauvaises réponses qui seront comptées. Si points > 0, c'est cette valeur qui sera utilisée pour le score.
  • dans tous les cas il faut affecter un nombre de points aux bonnes réponses pour les identifier et calculer le bon score.
  • images (optionnelle): vous pouvez associer une image qui sera affichée selon le template qui sera sélectionné
  • hauteur de l'image : Défini la taille de l'image par la hauteur, attention aux images trop larges.
  • mélanger les propositions : la liste des caractère alphanumériques ou expression sera mélangée à chaque lancement du quiz..
  • directive (optionnelle): un texte court pour indiquer à l'utilisateur ce qu'il doit faire
  • propositions : Liste de caractères alphanumériques ou expressions cliquable. Chaque caractère ou expression doit être séparé par un tiret "-". Pour forcer un retour à la ligne dans la suite de caractère ou expression il suffit de doubler le séparateur.
  • bouton "X" :   Efface le champ proposition
  • bouton "@" : Affecte au champ proposition une des caractères alphabétiques
  • bouton "#" : Affecte au champ proposition une des caractères numériques
  • bouton "@#" : Affecte au champ proposition une liste des caractères alphabétiques et numériques
  • Template : le slide propose plusieurs dispositions :
    • les trois traits courts et  noirs représentent les mots entre accolades repris dans la question
    • la forme rouge représente la ou les réponses cliquées par l'utilisateur
    • La forme bleue représente l'image
    • les trois traits longs représentent la proposition de caractères alphanumériques

Propositions :

Liste de bonnes réponses. Chaque réponse doit avoir un nombre de points supérieure à zéro.
Il est possible, bien que pas conseillé, d'ajouter des mauvaises réponses avec un nombre de points négatif.

Le poids permet de réordonner la liste sans devoir faire des couper / coller. Si l'option "Mélanger les propositions" a été cochées l'ordre des réponses à peu d'importance.
