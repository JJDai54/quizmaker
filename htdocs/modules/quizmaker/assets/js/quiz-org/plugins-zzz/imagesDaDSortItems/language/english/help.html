imagesDadSortItems: Sort images

Description:

This question offers several images that must be ordered according to the criterion defined in the question.
You have to move the images to insert or exchange them (see option)
The number of proposals is limited to 8.
adjust the height of the images according to the number of proposals to that the slide remains clear.

It is necessary to define the number of points higher in the body of the question.

Objective:

Replace the images or photos in the order defined in the question.

Options:

  • height: this is the height images to sort (see the proposals more bas).
  • title position: the title of the images or photos can be placed above or below see no All. It will also be used in the "title" property and will appear on mouseover.
  • fashion movement:
    • Exchange images: the source image and the destination image are exchanged, and the other images remain in their place.
    • Insert images without position indicator: the source image is inserted before the image destination, the other images are shifted.
    • Insert images with a position indicator: image source is inserted before the image destination, the other images are shifted and when moving one position indicator is displayed between images.

Proposals:

Each proposal includes:
  • Deletion: box has checking at the start of the line allows you to delete the proposition properly, image included.
  • an image: Used in preference for images with the same size format at the risk of disrupt the display. Do not use images that are too heavy.
  • a title (optional): If it is specified it will also appear when hovering over the mouse even if the "Display image titles" option is "None". The spaces and the "/" character will be replaced by a newline for a best presentation.
  • weight: it defines the order suggestions to find.
It is possible to define up to 8 proposals. Pay attention to the weight images, check the maximum weight allowed by your environment (module - Xoops preferences and hosting),
