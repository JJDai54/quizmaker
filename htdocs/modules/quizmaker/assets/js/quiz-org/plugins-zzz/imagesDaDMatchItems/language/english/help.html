imagesDadMatchItems: Put the images in the right place

Description:

This slide presents a list of images and a list of expressions both in disorder.
In some layout, a substitute image may be presented, see the proposals and the layout below.

Objective:

It you have to replace each image with the correct expression..

Options:

  • images height : Height images to put back in the right place.
  • height of silhouettes : Height placeholder images.
  • optional directive: text to specify what the user should do.
  • layout : several arrangements are possible.
    • first layout represented by a simple square, images and expressions are mixed and displayed together
    • others layouts: Images and expressions are separated into two areas,  And a placeholder image is displayed with each expression.

Proposals:

Each proposal includes:
  • Deletion: box has checking at the start of the line allows you to delete the proposition properly, images included.
  • an image: Used in preference for images with the same size format at the risk of disrupt the display. Do not use images that are too heavy.
  • an expression: each image must be replaced with the correct expression.
  • weight: it defines the order suggestions to find.
  • points: number of points for each couple. It must be equal to or greater than "1".
  • placeholder image: This image is used in layouts other than the first. It is displayed with the expression, and will be deleted after association of an image with an expression.
It is possible to define up to 12 proposals. Pay attention to the weight images, check the maximum weight allowed by your environment (module - Xoops preferences and hosting),
