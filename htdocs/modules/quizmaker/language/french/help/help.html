Help: Quizmaker

Description

The Quizmaker module can be used to modules in XOOPS

Install/uninstall

No special measures necessary, follow the standard installation process and extract the quizmaker folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Features

The ModuleBuilder module continues to expand, to get to the conditions to create modules, more and more sophisticated.
For this reason, I invite all developers to report and send in svn any changes or additions to this module, so that we can jointly contribute to the development
