Installation / Désinstallation

Tout d'abord, il est nécéssaire d'installer le framework  https://github.com/JJDai54/janus

Ce framework regroupe toute une série de fonctionalités utilisées par les modules que j'ai développés.
Une fois le frameworl installé, le module s'installe comme tout module sous xoops, voir l'ai de xoops pour l'installation de module.

Récupérer l'archive du module ici : https://github.com/JJDai54/quizmaker
ET l'installer comme tout module Xoops.
