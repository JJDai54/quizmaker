Plugins (Slides / plugin)

L'onglet "Plugins" permet de se faire une idée de ce qu'il est possible faire avec "quizmaker".

Les termes termes suivants sont souvent  utilisés indiféremment pour désigner le même objet, toutefois voici ce qu'il désignent plus précicément :.
  • Plugin  : défini l'ensemble des fichiers utiles à un plugin.
  • Slide : Défini le formulaire affiché avec tous les éléments nécessaire à son exécution, entête, question, proposition,  boutons de navigation, bouton d'aide, ...
  • Question : désigne la requête à laquelle doit répondre l'utilisateur et comprent tous les éléments nécéssaires à sa présentation, la question, les propositions, les images, ...
