Créer une catégorie

Dans l'onglet "Catégorie" cliquer sur "Ajouter une catégorie".

Description des options principales
Nom de la catégorie
Description
 Elle sera affichée côtéé user en dessous du choix de la catégorie.
Thème
 Plusieurs options de coloration des quiz est proposée. Cette valeur est reprise par défaut lors de la création d'un quiz, mais elle pourra âtre modifiée pour chaque quiz.
Poids
 Ordre d'afichage des catégories
Permissions
 Les permissions d'une catégorie sont applicable à tous les quiz de celle-ci.
Ells sont reprises dans l'onglet "Permissions" dans lequel sont définies également les autorisations globales à paramètrer en premier.

Dans l'onglet "Plugins" il est possible d'importer des exemples de quiz. Lors de l'importation ces quiz sont créer dans la catégorie "Exemples de quiz par plugin" qui est ajoutée automatiquement si elle n'existe pas encdore.
Evitez d'untiliser cette catégorie pour vos propre quiz.

Il est conseillé de créer une catégorie "test" pour mettre au point les quiz qui une fois vzalider pourront être basculé dans une autre catégorie.
Lors de l'importation d'un nouveau quiz, une nouvelle catégorie pourra être crée si l'option "Catégorie d'origine" est choisie (voir le chapitre "Importation de quiz").
