Ajouter des questions aux quiz

Le formulaire d'ajout de questions est composé de trois partie:
  • Les paramètre généraux commun à tous les type de question
  • Les options spécifiques sui peuvent être différentes pour chaque type de question
  • Les propositions de réponses qui elles aussi peuvent être différentes selon le type de question.
    Il peut également y avoir plusieurs groupes de propositions avec des paramètres différents.
Pour les deux dernières, il faudra se référer à l'aide du plugin pour en savoir d'avantage.

Options propres au plugin - La liste n'est pas exhaustive.
Points
 Nombre de points global en cas de bonne réponse.
Important il faut faire un choix entre deux cas de figures :
  • un nombre de points global qui sera le même quelque soit la réponse quand elle correcte,
  • un nombre de points dépendant du nombre de points attribués à chaque proposition.
Exemple : cas d'un slide "choiceSimple" avec l'option réponses multiples. Cela va générer une liste de case à cocher avec plusieur réponses valides.
Supposons 7 propositions dont 3 ont un nombre de points égal à 2 chacune et les autres 0.
Si le nombre de points global est de 5 et que toutes les cases à cocher sont correctes le score sera de 5
Si le nombre de points global est de 0 et dque toutes les cases à cocher sont correctes, le score sera de 3 x 2 = 6.
image
 Certain plugin permette d'afficher ne image pour agrémenter l'interface. L'image peut bien sur être en rapport direct avec la question.
Hauteur de l'image
 L'imge peut être redimentionnée en hauteur, la largeur sera automatiquement ajustée en fonction.
Mélanger les propositions Certains Slide proposent de mélanger les propositions. A chaque ecécution du quiz, les propositions seront mélangée, il ne sera pas possible dans ce cas de se référer à la numérotation.
Disposition
 Cet option est présentée sous forme d'icones qui représentes succintement la disposition des éléments du slide.
Important : dans certains slide, la disposition est dépendante du nombre de groupe défini par exemple, attention de faire le bon choix.
Directive
 Texte suplémentaire affiché dans la zone des proposition pour guider l'utilsateur, par exemple pour lui expliquer qu'il faut cliquer et déplacer une image.



   
Propositions - La liste n'est pas exhaustive.
Mot ou expression
 REprésente une proposition bonne ou mauvaise. Dans certain plugin ce pourra être une image.
Poids
 Ordre de présentatino. Il peut être modifier  par le paramètre "Mélanger les propositions" sauf dans certain cas ou l'ordre est important comme dans le plugin "sortItems" ou l'objectif est de retrouver l'ordre défini par le poids.
Points
 Certain plugins propose un nombre de points pour chaque proposition.
Important, ce nombre de points permet de définir les bonnes et les mauivaises rpponses.
Cependant si le nombre de points global est supérieur à zéro c'est lui qui sera utilisé pour calculé le score sinon ce sera en général, mis pas forcément, la somme des points des propositions.
Voir l'aide de chaque plugin pour en savoir plus.
Titre
 Utilisé par fois en association avec des images
Image
 Certains plugins propose une image à la place d'un mot ou d'une expression, exemple de "dadSortItems"
Couleur
 Le plugin "imageColor" propose de choisir une couleur, plutôt q"un mot ou une expression.
Dans l'exemple à importer Il utilise notamment un slide de type "pageInfo" pour présenter une image avec des couleurs., et dans un slide plus loin il faut retrouver ces couleurs.

Dans tous les cas il est nécessaire d'en référer à l'aide intégrée du plugin.
